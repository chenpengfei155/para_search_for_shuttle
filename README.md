# para_search_for_shuttle
# Shuttle 方案参数搜索算法

这是一个 `shuttle` 方案参数搜索算法，其功能是对输入参数进行合法性检查，并在参数合法的情况下，计算公钥尺寸、签名尺寸、组合尺寸以及对应的安全性估计。

## 输入参数

算法的输入参数包括：

- 模数 $q$
- 维度 $n$
- 矩阵行数 $m$
- 矩阵列数 $\ell$
- 离散高斯分布的标准差 $\sigma$
- 参数 $\alpha_h$

## 功能说明

算法首先对输入参数进行合法性检查，包括：

1. 维度 $n$ 必须取为安全参数的 2 倍；
2. 模数 $q$ 必须为 NTT 素数；
3. 标准差 $\sigma$ 必须满足：
   
   $$
   \sigma > 0.5
   $$

4. 参数 $\alpha_h$ 必须为 2 的幂次。

在参数满足上述条件后，算法将依据给定公式计算：

- 公钥尺寸（Public Key Size）
- 签名尺寸（Signature Size）
- 组合尺寸（Combined Size）

并将计算结果与预期值进行比较。

最后，算法会输出对应的安全性估计结果，用于评估参数配置的安全强度。
